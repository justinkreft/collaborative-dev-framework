### For Your First Project
1. [README.md](README.md) - Understand what this is
2. [EXAMPLE_SETUP.md](EXAMPLE_SETUP.md) - Choose setup option (lightweight/copy/symlink)
3. [SOCIAL_CONTRACT.md](SOCIAL_CONTRACT.md) - Establish working relationship and communication rules
4. [GOVERNANCE.md](GOVERNANCE.md) - Pick principles and templates to use

6. Start building, capture lessons

2. Review your project's LESSONS.md
3. Extract generic patterns
4. Update relevant framework doc
5. Commit improvements
